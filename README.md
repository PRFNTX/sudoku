Run Project for Dev:
    1) $ npm install
    2) $ npm start

Build and run
    1) $ npm run build

Add new Puzzles: 
    1) open /src/boards/boards.js
    2) add a new item to the 'boards' array:
        a) each item is an array
        b) each array contains 9 sub-arrays
        c) each sub-array contains 9 string elements either "[1-9]", or ""
